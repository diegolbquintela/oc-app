## Usage

```
change config.env file in config folder
```

```
 Npm Install
 cd client npm install
 cd ..
 cd server npm install
 cd ..


 1) Run the Backend
 cd server
 npm start

 2) Run the Frontend only
 cd client
 npm start

 # Build client
 cd client
 npm run build

 # Prod
 npm start
```
